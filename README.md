# Sales Performance Analysis & Profitability Insights

## Objective
Analyze sales and profitability data to uncover trends and provide actionable business insights.

## Dataset
Superstore Sales Dataset (public dataset)

## Business Questions
- Which regions generate high revenue but low profit?
- Which categories contribute to losses?
- How do customer segments impact revenue?
- What actions can improve profitability?

## Tools Used
- SQL
- Python
- Excel
- Power BI

## Key Insights
Insights available in the insights folder.

## Project Structure
- data → dataset
- sql → analysis queries
- python → data analysis
- powerbi → dashboards
- insights → recommendations
